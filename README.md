# AutoSage App Using Gemini Flash

## Project Description
AutoSage App is an AI-based web application developed using Python and Streamlit.
The application demonstrates the integration of Google Gemini Flash to provide
vehicle-related information through a simple and user-friendly interface.

## Features
- Upload vehicle image
- Display vehicle details
- Interactive Streamlit UI
- AI integration workflow

## Technologies Used
- Python
- Streamlit
- Google Gemini API
- REST API
- dotenv
- Pillow

## How to Run

1. Install required libraries

2. Run the application
